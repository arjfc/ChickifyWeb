Go into the project folder
After cloning, move into the folder by typing:

cd repository-name


Install the dependencies

The project has some libraries and packages it needs in order to run.

To install them, type:

npm install

Or if you are using Yarn:

yarn install


Start the project

To run it in development mode, type:

npm run dev


(or yarn dev).

If you want to build it for production, type:

npm run build
npm start

Open the project in your browser

By default, the project will run on:

http://localhost:3000 or :5173
(or whatever port is specified).
